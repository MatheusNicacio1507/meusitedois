Administração de Banco de Dados

São os administradores ou administradoras de banco de dados, também chamados de profissionais DBA. Mantém uma base de dados robusta, otimizada e completa para consulta e análise de grandes quantidades de dados (Big Data), além de gerenciar, manter, instalar e monitorar os dados corporativos, sua integridade, eficiência e disponibilidade

Conhecimentos Necessários

Os conhecimentos necessários são: exigido que você tenha graduação em sistemas da informação ou ciência da computação, assim como conhecimentos de linguagens de banco de dados, como SQL, XQuery, OQL, GraphQL e LINQ.
